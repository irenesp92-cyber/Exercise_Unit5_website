¿Qué es Git?

Git es una herramienta de control de verisones local que almacena instantáneas en lugar de una lista de cambios

Las ramas

La ramificación nos permite trabajar en varias líneas de desarrollo

Se puede tener una rama principal de código y varias más para funcionalidades.

Trabajar con ramas es como trabajar con repositores distintos pero con un tronco común, que sería la rama principal o main.
